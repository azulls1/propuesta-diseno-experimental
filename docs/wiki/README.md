## Cómo usar esta wiki

1. **Empieza por** [[00-tldr]] para tener la visión general en 1 minuto.
2. **Luego ve a** [[01-objetivo]] y [[02-descripcion-detallada]] para entender qué pide el maestro.
3. **Después lee los requisitos uno por uno**: [[03-req-1-hipotesis]] → [[04-req-2-metodologia]] → [[05-req-3-train-val]] → [[06-req-4-comparacion]].
4. **Antes de escribir**, revisa [[07-rubrica-explicada]] (sobre todo el Criterio 3, que pesa 40%) y [[08-glosario]].
5. **Para escribir el documento**, sigue [[09-plan-redaccion]].
6. **Antes de entregar**, pasa por [[10-checklist-final]].
7. **Si tienes dudas pendientes**, anótalas en [[11-decisiones-pendientes]].
